## flutter_material_ui

Let's start...

## Getting started

To use this plugin, add flutter_converter as a dependency in your pubspec.yaml file.

#### Example

```dart
import 'package:flutter/material.dart';
import 'package:flutter_converter/flutter_converter.dart';

void main() {
  runApp(
    MaterialApp(
      home: Scaffold(
        body: SafeArea(
          child: Text(FC.Month('May')),
        ),
      ),
    ),
  );
}
```
